What is JSON?

JSON (JavaScript Object Notation) is a lightweight data-interchange format that is easy for humans to read and write, and easy for machines to parse and generate.

Why is JSON so popular?

  • Simple and human-readable syntax
  • Supported by almost every programming language
  • Lightweight compared to XML
  • Ideal for APIs and web services

JSON basic structure

JSON is built on two main structures:

  • Objects ‚Äî key/value pairs
  • Arrays ‚Äî ordered lists of values

  "active": true

Common JSON errors

  • Missing commas
  • Using single quotes instead of double quotes
  • Trailing commas
  • Unquoted keys
